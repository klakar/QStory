QStory

Welcome, you brave person...

This plugin helps you write a map story for your web maps. The plugin will only create the story and the nessesary dependency files. You will then need to point the plugin towards a web map you already have. You can create your own in QGIS with the qgis2web plugin.

  1. Write story on Code tab
  2. Update the location and zoom using the map canvas (pan/zoom then click button)
  3. Try different Themes (only a few to start)
  4. Save your story to a *.qstory file (optional)
  5. Find a web map template to apply the story to (use qgis2web)
  6. Generate the new story map (same location as the template)

Known Limitations

  • Current themes are hard coded to JS-libraries (i.e. OpenLayers or Leaflet).
  • For current templates it is assumed you have a map "div" with the id "map".
  • You can't reorder the pages in the plugin.
  • I will only create templates I like and am able to build... (probably very few for Leaflet)

Note that most improvements you will want, can probably be done with JavaScript and CSS. If you want to create your own "theme", just modify the files in the qstory folder in your web map directory.

If you have suggestions, want to contribute with JS/CSS themes, or have found something wrong (remember this is experimental), use GitHub to report issues or make a pull request.
